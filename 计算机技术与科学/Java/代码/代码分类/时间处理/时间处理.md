时间工具类中的方法：
1. 判断是否是今天
   转换为相同的日期格式后用String的equals进行比较
   ```java
   // 判断是否是今天  
public static boolean isNow(Date date) {  
   Date now = new Date();  
   SimpleDateFormat sf = new SimpleDateFormat("yyyyMMdd");  
   String nowDay = sf.format(now);  
   String day = sf.format(date);  
   return day.equals(nowDay);  
}
```
