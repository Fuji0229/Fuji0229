## 1.HTTP基础认证和表单登录认证

HTTP基础认证：
	开启方式：http.httpBasic();
	认证方式：通过HTTP的消息头携带用户名和密码进行登录。
HTTP基础认证比较简单，没有定制的页面，一般与表单登录认证结合使用。

表单登录认证：
	开启方式：http.forFormLogin();
	开启后默认会提供一个默认的登录界面
