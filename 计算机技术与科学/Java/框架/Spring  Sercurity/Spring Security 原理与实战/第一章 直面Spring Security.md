为什么使用Spring Security？
1.日常开发的业务系统或多或少有安全性的技术要求，最常用的就是用户认证和访问授权。
2.Spring Security在日常开发过程中可以和Spring boot项目无缝继承，而且是Spring cloud 等综合性开发框架的底层基础框架之一。

Spring Security提供的场景
1.用户信息管理
2.用户认证
3.权限控制
4.跨站点请求伪造保护
5.跨域支持
6.全局安全方法
7.单点登录
